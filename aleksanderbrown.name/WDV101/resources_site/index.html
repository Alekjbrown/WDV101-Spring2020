HTML Online Resources

There are many resources available across the web to learn to write your own websites in HTML and CSS. some of them are listed below.

  • W3 Schools - CSS Framework

    • W3 Schools has many options for tutorials on website building. This particular one focuses on building one using a CSS framework for styles.

  • W3 Schools - Basic Website

    • This W3 Schools tutorial has a basic process for building a website. It walks you through draft design, to basic html, and how to put that content into a layout

  • Website How to with

    • Code Academy offers a free course step by step to building a website. It begins by explaining what HTML and CSS are and steps through how to use them while you build a website within their website.

  • Microsoft docs - simple website building

    • Microsoft docs hold a wealth of technology information. This includes a tutorial on utilizing HTML, CSS, and even Javascript. These languages are used to an extent with a number of other Microsoft coding languages. Therefore, it is imperative to them that you know what they are and how they use them.

  • Open Classrooms - Build your first web pages

    • Open Classrooms offers a full tutorial from start to finish to build a website with HTML and CSS. If you don't need the full step by step you can also skip around using content links to specific topics.
